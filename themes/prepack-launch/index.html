PRE.PACKED

A PrePacked fork of Pico -- a stupidly simple and blazing fast flat flie CMS, that just got a lot easier...

Pico is simple and easy to use flat file content management system written by Gilbert Pellegrom the founder of Dev Studios the guys that created the famous Nivo Slider. PrePacked is a fork of Pico that we modified and packed with some of our favorite plugins then added some of our own to make it simple and easy for us to build and deploy Pcio based websites. This is a preview release read about the Features here, or fork it here on Github.

Preview release you can follow or contribute on Github

PICO Extended (V0.7)

We take the latest version of Pico V0.7, modify / extend the pico library so that we can customize any way we choose with little or no fear of breaking our apps on future releases, this gives us the flexibility to modify and customize pico functions that we choose.

Prepacked Pico Plugins

We took some of our favorites Pico Plugins and added them as git modules to our fork, this saves us the trouble of having to go searching and downloading plugins that we use on a regular basic all the time. If you have a plugin that you would like to see added to this fork, post it in the official pico plugin wiki and we will find it.

Custom Meta Values

Create and use custom meta values in Pico using the latest version of Advanced Meta, a plugin created to make it easier to create and access custom meta page values in Pico V0.7.

Page Layout(s)

Custom pages lets you specify and use custom HTML templates for pages, but we did not stop there we created themes that leverage Twig's template inheritance feature by using blocks to change only the parts of the base templates you need. You can read more about template inheritance here in the Twig docs.

Dynamic Page Galleries

Quickly create galleries for your page by adding a gallery folder to your content dir and placing folder(s) inside it named after the page slug or file_name of the page you want the gallery to appear, place all of your images inside of that folder then add another folder named thumbs to it. Dynamic Galleries will locate the folder, create the thumbnails inside and add them to the your theme variables for you to access.

Starter Theme

No fork would ever be complete without something to get you up and running quickly, so if you would like to test out some of the features we added go to your Pico config file and change your theme config value to theme-starter, now go to you pico site and check it out.
